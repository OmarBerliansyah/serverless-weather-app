# SweaterWeather

Welcome to **OmarCast**, a simple, modern, and functional weather application built with Vanilla JavaScript. This project displays current weather data and a 5-day forecast from anywhere in the world through a clean and aesthetic interface.

🌐 **[View Live Demo](https://sweater-weather.vercel.app)** ➡️

## Key Features

This application does more than just display the temperature. Here's everything it can do:

- **Global Weather Search**: Get the latest weather data from any city in the world.

- **Detailed View**: Weather information is broken down into several categories:
  - **Atmosphere**: Feels-like temperature, air pressure, humidity, and the day's min/max temperatures.
  - **Wind**: Wind speed, direction, and gusts.
  - **Daylight**: Sunrise and sunset times.
  - **Visibility**: Viewing distance and cloud cover percentage.

- **5-Day Forecast**: See the weather forecast for the next 5 days, complete with icons and temperatures.

- **Geographic Info**: Hover over the info icon to see Latitude and Longitude coordinates, with a direct link to Google Maps.

- **Responsive Interface**: Designed to look great on both desktop and mobile devices.

## Technologies Used

This project is built with fundamental web technologies, without any complex frameworks. Pure and simple.

- **HTML5**: For content structure.
- **CSS3**: For styling, layout, and the "glassmorphism" aesthetic.
- **Vanilla JavaScript**: For all logic, interactivity, and fetching data from the API.
- **OpenWeatherMap API**: As the source for all weather and forecast data.

## How to Run This Project on Your Computer

Want to try or modify this code? Follow these steps:

### 1. Clone This Repository

```bash
git clone https://github.com/OmarBerliansyah/serverless-weather-app.git
cd your-repository-name
```

### 2. Get a Free API Key

1. Go to [OpenWeatherMap](https://openweathermap.org/api) and sign up for a free API Key.
2. Copy the API key.

### 3. Enter the API Key

1. Open the `app.js` file.
2. Find the line `const apiKey = '...';` and replace the content with the API key you just obtained.

### 4. Run the Project

Since this is a static project, you just need to open the `index.html` file in your favorite browser. For the best development experience, use the "Live Server" extension in Visual Studio Code.
